# Portfolio técnico

## Proyectos individuales

### `minishell` ― Intérprete de comandos

Implementación del núcleo funcional de un shell Unix con la gestión del ciclo REPL, el control de procesos y el tratamiento de señales POSIX. Incluye expansión de metacaracteres, redirecciones, *pipelines* y ejecución de mandatos internos (`cd`, `umask`, `time`, `read`).

**Asignatura**: Sistemas Operativos

**Curso**: 24/25

**Lenguajes**: `C`

**Herramientas**: `Linux`, `GCC`, `Makefile`, `flex`, `bison`, `VSCode`, `Git`

> Repositorio privado ― disponible bajo petición

---

### `map` ― Herramienta de procesamiento *batch* en *cluster*

Implementación de una herramienta de Big Data basada en la fase *map* de *MapReduce*. El sistema ejecuta en paralelo un programa *mapper* sobre particiones de un fichero, distribuyendo el trabajo entre múltiples nodos.

**Asignatura**: Sistemas Distribuidos

**Curso**: 24/25

**Lenguajes**: `C`

**Herramientas**: `Linux`, `GCC`, `Makefile`, `VSCode`, `Git`

> Repositorio privado ― disponible bajo petición

### Compilador para *MyJS*

Implementación de un procesador de lenguajes que analiza programas en el dialecto educativo *MyJS*. El sistema realiza los análisis léxico, sintáctico y semántico, gestiona los errores en cada fase y genera el árbol sintáctico en formato VAS para su visualización. Incluye un conjunto de pruebas con ficheros de ejemplo y salidas generadas automáticamente.

**Curso**: 24/25

**Lenguajes**: `Java`

**Herramientas**: `Java`, `JUnit`, `VSCode`, `Git`

### `Parkings EMT` ― Integración semántica de datos abiertos

Transformación de un dataset de aparcamientos de *EMT Madrid* siguiendo las buenas prácticas de *Linked Data*. Incluye la definición de la ontología, la generación de RDF, la reconciliación de entidades con *Wikidata* y la carga del grafo en *GraphDB* para su consulta con SPARQL, así como el desarrollo de una aplicación para buscar y filtrar parkings.

**Asignatura**: Semantic Web, Linked Data and Knowledge Graphs

**Curso**: 25/26

**Lenguajes**: `Python`, `HTML`

**Herramientas**: `Chowlk`, `YARRRML`, `RMLMapper`, `OpenRefine`, `GraphDB`, `Python`, `VSCode`, `Git`

> Repositorio público ― [`sw/HandsOn/Group20`](https://github.com/lauugr/sw/tree/master/HandsOn/Group20)

### `BeerSp` ― Aplicación social Flutter para explorar cervezas y locales

Implementación de una aplicación social multiplataforma orientada al turismo gastronómico y al descubrimiento de cervecerías siguiendo los estándares de Ingeniería del Software. Incluye registro y autenticación, perfiles de usuario, *feed* social, sistema de amistad, gestión de degustaciones (búsqueda, alta, valoración y comentarios) y listado de cervezas más valoradas.

**Asignatura**: Ingeniería del Software II

**Curso**: 25/26

**Lenguajes**: `Dart`, `C++`

**Herramientas**: `Flutter`, `Dart`, `Firebase Auth`, `Firestore`, `Firebase Storage`, `Android/iOS/Web`, `C++`, `Git`
